wololo wake-on-lan web interface

message here

wake directly using an HTTP request

/wake/<DeviceName> returns a JSON object 

{
  "success":true,
  "message":"Sent magic packet with MAC 34:E6:D7:33:12:71 on Broadcast IP 192.168.1.255:9",
  "error":null
}

success
true if the packet was sent, false otherwise
message
action result message
error
JSON representation of the error
